# Grid

The `Grid` component allows you to create a flexible grid layout with customizable gap between items. It is useful for displaying content in a structured and responsive manner.

<Canvas of={GridStories.Default} />

## Usage

To use the `Grid` component, import it and pass the necessary props:

```tsx
import { Grid } from '@aj-codes2020/ui-kit';

const gridChildren = (
  <div className="grid-container">
     <div className="grid-d-6 grid-t-3 grid-m-12" style={{ backgroundColor: '#f0f0f0', height: '50px'}}>Item 1</div>
     <div className="grid-d-6 grid-t-3 grid-m-12" style={{ backgroundColor: '#c0c0c0', height: '50px' }}>Item 2</div>
     <div className="grid-d-6 grid-t-3 grid-m-12" style={{ backgroundColor: '#f0f0f0', height: '50px'}}>Item 3</div>
     <div className="grid-d-6 grid-t-3 grid-m-12" style={{ backgroundColor: '#c0c0c0', height: '50px'}}>Item 4</div>
   </div>
);

<Grid className="">
  {gridChildren}
</Grid>
```

## Class Selectors

You can customize the `Grid` component using CSS. The following class names are available for customization:

`grid-container`: The main container for the grid.\
`grid-1` to `grid-12`: The class names for column widths.

### Breakpoints:

`grid-m-*`: (min-width: 320px)\
`grid-t-*`: (min-width: 480px)\
`grid-d-*`: (min-width: 768px)

### Class Selectors and Their Widths

`grid-*-2`: 16.66%\
