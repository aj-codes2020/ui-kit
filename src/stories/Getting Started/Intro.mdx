## About

This UI Kit list a set of components that streamline the process of developing websites and applications.

The components are grouped into four main categories:

- **Layout**: These components help you structure your web pages for responsive design.
- **Navigation**: This category includes components that handle navigation for your users.
- **Page Elements**: This category includes components that help you create engaging and interactive content.
- **UI Elements**: These components enhance the interactivity and usability of your interface.

Each component in the UI Kit comes with documentation, including:

- **Component Description**: An overview of what the component does and how it can be used in your projects.
- **Use of Code**: Practical example showing how to implement the component.
- **Props**: A list of properties that the component accepts, allowing you to customize its behavior and appearance.
- **Class Selectors**: CSS class selectors that you can use to further customize the component's styling to match your project's design needs.

Feel free to explore and make use of this Kit to bring your web projects to life.

## Add the Kit as a dependency

You can add the Kit to your project as a dependency. To install the Kit through npm, run:

```bash
  npm install @aj-codes2020/ui-kit
```

## Using the Kit

Once installed, you can start using the components in your project. Here is an example of how to use the `Button` component:

```jsx
  import { Button } from '@aj-codes2020/ui-kit';

<Button variant="primary">Click Me</Button>
```
