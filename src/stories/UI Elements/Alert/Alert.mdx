# Alert

The `Alert` component is used to display important messages to the user, such as informational, success, warning, and error alerts.

## Usage

To use the `Alert` component, import it and pass the necessary props:

```tsx
import { Alert } from '@aj-codes2020/ui-kit';

<Alert type="info" message="This is an informational alert" />;
<Alert type="success" message="This is a success alert" />;
<Alert type="warning" message="This is a warning alert" />;
<Alert type="error" message="This is an error alert" />;
```

## Class Selectors

You can customize the `Alert` component using CSS. The following class names are available for customization:

`alert`: The main container for the alert.\
`alert-info`: The container for an informational alert.\
`alert-success`: The container for a success alert.\
`alert-warning`: The container for a warning alert.\
`alert-error`: The container for an error alert.
