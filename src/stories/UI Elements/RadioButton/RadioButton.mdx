# Radio Button

The Radio Button component allows users to select one option from a group of options. Each radio button in a group is mutually exclusive, meaning only one button can be selected at a time.

<Canvas of={RadioButtonStories.Default} />

## Usage

To use the Radio Button component, import it and provide an array of options with `label` and `value` properties. 

```tsx
import { RadioGroup } from 'path-to-your-components';

const options = [
  { label: 'Option 1', value: 'option1' },
  { label: 'Option 2', value: 'option2' },
  { label: 'Option 3', value: 'option3' },
];

<RadioGroup name="exampleGroup" options={options} onChange={(value) => console.log(`Selected value: ${value}`)} />;
```

## Customization

You can customize the Button component using CSS. The following class names are available for customization:

`radio-button`: The main container for each radio button.\
`radio-button input[type='radio']`: The radio button input (hidden by default).\
`radio-custom`: The custom styled radio button.\
`radio-button input[type='radio']:checked + .radio-custom`: The checked state of the custom radio button.
