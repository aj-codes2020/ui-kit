# Progress Bar

The Progress Bar component is used to visually represent the progress of a task or an operation. It can be displayed as either a linear or circular progress bar.

<Canvas of={ProgressBarStories.LinearProgress} />

## Usage

To use the Progress Bar component, import it and provide the necessary properties such as `value`, `max`, `type`, `color`, and `size`.

```tsx
import { ProgressBar } from 'path-to-your-components';

const App = () => (
  <>
    <ProgressBar value={50} max={100} type="linear" color="primary" size="medium" />
    <ProgressBar value={75} max={100} type="circular" color="success" size="large" />
  </>
);
```

## Customization

You can customize the Button component using CSS. The following class names are available for customization:

`progress-bar`: The main container for the linear progress bar.\
`progress-bar-fill`: The fill element of the linear progress bar.\
`progress-circle`: The main container for the circular progress bar.\
`progress-circle-bg`: The background circle of the circular progress bar.\
`progress-circle-fg`: The foreground circle of the circular progress bar.\
`progress-bar.primary`: The primary color style for the progress bar.\
`progress-bar.secondary`: The secondary color style for the progress bar.\
`progress-bar.success`: The success color style for the progress bar.\
`progress-bar.warning`: The warning color style for the progress bar.\
`progress-bar.danger`: The danger color style for the progress bar.\
`progress-bar.small`: The small size style for the progress bar.\
`progress-bar.medium`: The medium size style for the progress bar.\
`progress-bar.large`: The large size style for the progress bar.
