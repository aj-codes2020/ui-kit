# Toggle Switch

The `ToggleSwitch` component allows you to create a switch that toggles between an "on" and "off" state, useful for settings and preferences.

<Canvas of={ToggleSwitchStories.Default} />

## Usage

To use the `ToggleSwitch` component, import it and pass the necessary props:

```tsx
import { ToggleSwitch } from '@aj-codes2020/ui-kit';

const App = () => {
  const handleToggle = (isOn: boolean) => {
    console.log(`Toggle Switch is now ${isOn ? 'On' : 'Off'}`);
  };

  return (
    <ToggleSwitch initialState={false} onToggle={handleToggle} className="" />
  );
};
```

## Class Selectors

You can customize the Toggle Switch component using CSS. The following class names are available for customization:

`toggle-switch`: The main container for the toggle switch.\
`toggle-switch.on`: The container when the toggle switch is in the "on" state.\
`toggle-switch.off`: The container when the toggle switch is in the "off" state.\
`toggle-thumb`: The thumb element inside the toggle switch.
