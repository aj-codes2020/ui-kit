# Badge

The `Badge` component is used to display status indicators or counters, useful for showing notifications or status information.

## Usage

To use the `Badge` component, import it and pass the necessary props:

```tsx
import { Badge } from '@aj-codes2020/ui-kit';

<Badge 
  color="default" 
  size="medium" 
  className=""
>
  Default
</Badge>
```

## Class Selectors

You can customize the `Badge` component using CSS. The following class names are available for customization:

`badge`: The main container for the badge.\
`badge.status`: The status type badge.\
`badge.counter`: The counter type badge.\
`badge.default`: The default color badge.\
`badge.primary`: The primary color badge.\
`badge.success`: The success color badge.\
`badge.warning`: The warning color badge.\
`badge.danger`: The danger color badge.\
`badge.small`: The small size badge.\
`badge.medium`: The medium size badge.\
`badge.large`: The large size badge.
