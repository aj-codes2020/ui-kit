# Text Area

The `TextArea` component is used for multiline text input, allowing users to enter large amounts of text.

<Canvas of={TextAreaStories.Default} />

## Usage

To use the `TextArea` component, import it and pass the necessary props:

```tsx
import { TextArea } from '@aj-codes2020/ui-kit';

const App = () => {
  const [value, setValue] = useState('');

  return (
    <TextArea
      value={value}
      onChange={setValue}
      placeholder="Enter text here..."
      rows={4}
      cols={50}
    />
  );
};
```

## Class Selectors

You can customize the `TextArea` component using CSS. The following class names are available for customization:

`textarea`: The main container for the textarea element.\
`textarea:focus`: The focused state of the textarea element.
