# Search Bar

The `SearchBar` component allows users to search and filter through a list of suggestions, providing an autocomplete functionality.

<Canvas of={SearchBarStories.Default} />

## Usage

To use the `SearchBar` component, import it and pass the necessary props:

```tsx
import { SearchBar } from '@aj-codes2020/ui-kit';

const App = () => {
  const handleSearch = (query: string) => {
    console.log(`Search query: ${query}`);
  };

  const suggestions = ['Apple', 'Banana', 'Cherry', 'Date', 'Elderberry', 'Fig', 'Grape', 'Honeydew'];

  return <SearchBar suggestions={suggestions} onSearch={handleSearch} placeholder="Search for a fruit..." />;
};
```

## Class Selectors

You can customize the `SearchBar` component using CSS. The following class names are available for customization:

`search-bar`: The main container for the search bar.\
`search-input`: The input field for the search bar.\
`suggestions-list`: The container for the list of suggestions.\
`suggestion`: Each individual suggestion item.\
`suggestion.active`: The active suggestion item.\
`suggestion:hover`: The suggestion item when hovered over.
