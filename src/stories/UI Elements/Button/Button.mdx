# Button

The `Button` component is a versatile element used to trigger actions in your application. It supports various sizes, colors, and styles.

<Canvas of={ButtonStories.Primary} />
<Canvas of={ButtonStories.Secondary} />

## Usage

To use the `Button` component, import it and pass the necessary props:

```tsx
import { Button } from '@aj-codes2020/ui-kit';

<Button 
  primary 
  label="Primary Button" 
  onClick={() => alert('Button clicked')} 
/>

<Button 
  label="Secondary Button" 
  onClick={() => alert('Button clicked')} 
/>
```

## Class Selectors

You can customize the `Button` component using CSS. The following class names are available for customization:

`button`: The main container for the button.\
`button--primary`: The primary style button.\
`button--secondary`: The secondary style button.\
`button--small`: The small size button.\
`button--medium`: The medium size button.\
`button--large`: The large size button.
