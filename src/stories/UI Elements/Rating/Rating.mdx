# Rating

The `Rating` component allows users to rate items such as products or services. Users can select a number of stars to represent their rating.

<Canvas of={RatingStories.Default} />

## Usage

To use the `Rating` component, import it and provide the necessary props:

```tsx
import { Rating } from '@aj-codes2020/ui-kit';

const App = () => {
  const [rating, setRating] = useState(3);

  return <Rating maxRating={5} value={rating} onChange={setRating} />;
};
```

## Class Selectors

You can customize the `Rating` component using CSS. The following class names are available for customization:

`rating`: The main container for the rating component.\
`star`: The individual star element.\
`star.filled`: The filled state of the star element.\
`star:hover`: The hover state of the star element.
