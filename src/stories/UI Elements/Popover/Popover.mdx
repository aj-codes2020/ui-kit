# Popover

The `Popover` component allows you to display additional content when users click on an element. It's useful for providing more information or options without cluttering the UI.

<Canvas of={PopoverStories.Default} />

## Usage

To use the `Popover` component, import it and pass the necessary props:

```tsx
import { Popover } from '@aj-codes2020/ui-kit';

const App = () => (
  <Popover content="This is a popover" position="bottom">
    <button>Click me</button>
  </Popover>
);
```

## Class Selectors

You can customize the `Popover` component using CSS. The following class names are available for customization:

`popover-container`: The main container for the popover.\
`popover-box`: The container for the popover content.\
`popover-top`: Applied when the popover is positioned at the top.\
`popover-bottom`: Applied when the popover is positioned at the bottom.\
`popover-left`: Applied when the popover is positioned at the left.\
`popover-right`: Applied when the popover is positioned at the right.
