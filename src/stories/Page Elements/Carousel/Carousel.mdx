# Carousel

The `Carousel` component allows you to create a slideshow of items that can autoplay and includes navigation options, providing a versatile way to display multiple pieces of content in a compact space.

<Canvas of={CarouselStories.ImageCarousel} />

## Usage

To use the `Carousel` component, import it and pass the necessary props:

```tsx
import { Carousel } from '@aj-codes2020/ui-kit';

const textItems = [
  <div style={{ padding: '2rem', backgroundColor: '#f0f0f0', height: '200px', display: 'flex', alignItems: 'center', justifyContent: 'center' }}>Text Item 1</div>,
  <div style={{ padding: '2rem', backgroundColor: '#c0c0c0', height: '200px', display: 'flex', alignItems: 'center', justifyContent: 'center' }}>Text Item 2</div>,
  <div style={{ padding: '2rem', backgroundColor: '#f0f0f0', height: '200px', display: 'flex', alignItems: 'center', justifyContent: 'center' }}>Text Item 3</div>,
];

<Carousel
  items={textItems}
  autoplay={true}
  autoplayInterval={3000}
  showNavigation={true}
  navigationType="dots"
  navigationPosition="bottom"
  className=""
/>
```

## Class Selectors

You can customize the `Carousel` component using CSS. The following class names are available for customization:

`carousel`: The main container for the carousel.\
`carousel-content`: The container that holds all the carousel items.\
`carousel-item`: The container for each carousel item.\
`carousel-item.active`: The container for the active carousel item.\
`carousel-navigation`: The container for the navigation controls.\
`carousel-navigation.bottom`: The container for the navigation controls when positioned at the bottom.\
`carousel-dots`: The container for the dot navigation controls.\
`dot`: The individual dot navigation control.\
`dot.active`: The active dot navigation control.\
`carousel-pictures`: The container for the picture navigation controls.\
`picture-thumbnail`: The individual picture navigation control.\
`picture-thumbnail.active`: The active picture navigation control.
