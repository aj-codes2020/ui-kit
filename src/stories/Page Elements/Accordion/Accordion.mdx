# Accordion

The `Accordion` component allows you to create expandable and collapsible sections, useful for presenting information in a compact and organized manner.

<Canvas of={AccordionStories.Default} />

## Usage

To use the `Accordion` component, import it and pass the necessary props:

```tsx
import { Accordion } from '@aj-codes2020/ui-kit';

const items = [
  { title: 'Section 1', content: 'Content for section 1' },
  { title: 'Section 2', content: 'Content for section 2' },
  { title: 'Section 3', content: 'Content for section 3' },
];

<Accordion items={items} />;
```

## Class Selectors

You can customize the `Accordion` component using CSS. The following class names are available for customization:

`accordion`: The main container for the accordion.\
`accordion-item`: The container for each accordion item.\
`accordion-title`: The title of each accordion item.\
`accordion-title.active`: The title of the active accordion item.\
`accordion-icon`: The icon displayed at the end of the accordion title.\
`accordion-content`: The content area of each accordion item.\
`accordion-content.active`: The content area of the active accordion item.
