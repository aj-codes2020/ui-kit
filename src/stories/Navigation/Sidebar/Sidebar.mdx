# Sidebar

The Sidebar component provides a layout with a sidebar and a header. The sidebar contains navigation links, and the header includes a hamburger menu to toggle the sidebar visibility on mobile devices. The sidebar also includes a title section above the links.

<Canvas of={SidebarStories.Default} />

## Usage

To use the Sidebar component, import it and pass the necessary props for `logoSrc`, `logoAlt`, `sidebarTitle`, `sidebarLinks`, `headerContent`, and `className`.

```tsx
import Sidebar from 'path-to-your-components';

const sidebarLinks = (
  <>
    <li><a href="/">Home</a></li>
    <li><a href="/link1">Link 1</a></li>
    <li><a href="/link2">Link 2</a></li>
    <li><a href="/link3">Link 3</a></li>
    <li><a href="/link4">Link 4</a></li>
  </>
);

<Sidebar
  logoSrc="https://via.placeholder.com/40"
  logoAlt="Logo"
  sidebarTitle="My Sidebar"
  sidebarLinks={sidebarLinks}
  headerContent={<div>Welcome, Jane Doe!</div>}
  className=""
>
  <div>
    <h2>Main Content</h2>
    <p>This is the main content area.</p>
  </div>
</Sidebar>;
