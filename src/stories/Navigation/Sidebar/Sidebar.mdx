# Sidebar

The `Sidebar` component provides a layout with a sidebar and a header. The sidebar contains navigation links, and the header includes a hamburger menu to toggle the sidebar visibility on mobile devices. The sidebar also includes a title section above the links.

<Canvas of={SidebarStories.Default} />

## Usage

To use the `Sidebar` component, import it and pass the necessary props:

```tsx
import { Sidebar } from '@aj-codes2020/ui-kit';

const sidebarLinks = (
  <>
    <li><a href="#">Home</a></li>
    <li><a href="#">Link1</a></li>
    <li><a href="#">Link2</a></li>
    <li><a href="#">Link3</a></li>
    <li><a href="#">Link4</a></li>
  </>
);

<Sidebar
  logoSrc="https://via.placeholder.com/40"
  logoAlt="Logo"
  sidebarTitle="Title"
  sidebarLinks={sidebarLinks}
  headerContent={<div>Header Content</div>}
  className=""
>
  <div>
    <h2>Main Content</h2>
    <p>This is the main content area.</p>
  </div>
</Sidebar>;
```

## Class Selectors

You can customize the `Sidebar` component using CSS. The following class names are available for customization:

`sidebar-layout`: The main container for the sidebar layout.\
`sidebar`: The sidebar itself. Use this class to style the sidebar.\
`sidebar.closed`: Applied when the sidebar is closed.\
`sidebar-logo`: The logo container in the sidebar.\
`sidebar-title`: The title section of the sidebar.\
`sidebar-title .close-menu`: The close menu button in the sidebar title section.\
`sidebar-nav ul`: The unordered list containing the sidebar links.\
`sidebar-nav li`: The list items containing the individual links.\
`sidebar-nav a`: The anchor tags for the sidebar links.\
`sidebar-nav li:hover, sidebar-nav li.active`: Styles applied to the sidebar links when hovered or active.\
`main-content`: The main content container that adjusts when the sidebar is open or closed.\
`main-content.full-width`: Applied to the main content container when the sidebar is closed.\
`header`: The header section of the layout.\
`header.full-width`: Applied to the header when the sidebar is closed.\
`hamburger-menu`: The hamburger menu button for toggling the sidebar.\
`header-content`: The container for additional header content.\
`body-container`: The container for the main body content.
