# Navbar

The `Navbar` component is used to display the main navigation header for your application. It includes elements for branding and customizable columns for additional content.

<Canvas of={NavbarStories.Default} />

## Usage

To use the `Navbar` component, import it and provide the necessary props:

```tsx
import { Navbar } from '@aj-codes2020/ui-kit';

<Navbar
  logoSrc="https://via.placeholder.com/40"
  logoAlt="Logo"
  logoTitle="Title"
  centerColumn={
    <>
      <a href="/center1">CenterLink1</a>
      <a href="/center2">CenterLink2</a>
    </>
  }
  rightColumn={ 
    <>
      <a href="/right1">RightLink1</a>
      <a href="/right2">RightLink2</a>
    </>
  }
  menuColumn={
    <>
      <a href="/menu1">MenuLink1</a>
      <a href="/menu2">MenuLink2</a>
    </>
  }
/>
```

## Class Selectors
You can customize the `Navbar` component using CSS. The following class names are available for customization:

`navbar`: The main container for the navbar.\
`navbar .logo`: The logo container in the navbar.\
`navbar .logo img`: The logo image.\
`navbar .logo h1`: The logo title.\
`navbar .column-center`: The center column of the navbar.\
`navbar .column-right`: The right column of the navbar.\
`navbar a`: The links in the navbar.\
`navbar .hamburger-menu`: The hamburger menu button (visible on mobile).\
`menu`: The container for the dropdown menu.\
`menu-header`: The header of the dropdown menu.\
`menu-header .logo`: The logo container in the dropdown menu header.\
`menu-header .logo img`: The logo image in the dropdown menu header.\
`menu-header .logo h1`: The logo title in the dropdown menu header.\
`menu-column`: The container for the menu column items.
